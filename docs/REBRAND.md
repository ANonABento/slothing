# Taida (怠惰) — Rebrand Plan

> Japanese for "sloth/laziness" — you're not lazy, you're efficient.

## Name Decision

**Taida** — intellectual, hidden meaning, 5 letters, memorable.

## Domain Strategy

### Phase 1: Umbrella (now)
- Launch under personal studio domain: `taida.{studio-domain}`
- Zero cost, validates the product

### Phase 2: Own Domain (when traction proven)
- Buy `taida.app` (~$15/yr) — likely available
- Backup: `gettaida.com` — confirmed available
- 301 redirect from umbrella → own domain
- SEO transfers within 2-4 weeks

## SEO Strategy

### Phase 1: Technical Foundation
- Next.js SSR/SSG (already have)
- Meta tags on every page
- sitemap.xml auto-generated
- Schema markup (JSON-LD)
- Google Search Console

### Phase 2: Content Funnel
- Free ATS scanner at `/ats-scanner` (no login)
- Target: "free ATS resume checker" (14k searches/mo)
- Blog posts: "How to pass ATS", "Best resume format for X"

### Phase 3: Authority
- Product Hunt launch
- Open source = GitHub backlinks
- Programmatic SEO: `/resume-templates/{role}`, `/ats-check/{company}`

## Brand Identity
- **Tagline:** "You're not lazy. You're efficient."
- **Vibe:** Anti-hustle, effortless, intellectual
- **Differentiator:** Local-first, open source, your data stays on your machine
