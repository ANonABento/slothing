# Competitor Analysis — AIApply.co

> Research date: April 2026

## Overview

- **Users:** 1.16M+
- **Pricing:** $29/mo toolkit + $39/100 auto-applies
- **Rating:** 4.8 stars (850 reviews) but BBB F-rated, Trustpilot flagged
- **Founded:** 2023, UK

## What They Do Well

### 1. Free ATS Scanner (Lead Gen Funnel)
- No account required
- Upload resume → get ATS score + feedback
- Scares users with low scores → converts to paid
- **This is their best growth hack**

### 2. Job Description Paste-to-Tailor
- Paste a JD, AI rewrites resume to match
- Simple, tangible, immediate value

### 3. Chat-Based Cover Letter
- Interactive chat to iterate on cover letters
- "Emphasize PM experience" → AI regenerates
- Their strongest feature UX-wise

### 4. Match Percentage Display
- Shows "99.8% match" on tailored resumes
- Gamification + confidence hook

### 5. Multi-Format Export
- PDF, DOCX, plain text
- LinkedIn import for onboarding

## What They Do Badly

### AI Quality
- Generic output — "spearheaded cross-functional initiatives" for everyone
- Hallucinated experience (invents work history)
- Cover letters over-optimize, read as inhuman

### Auto-Apply
- Spray-and-pray — wrong roles, wrong locations, wrong languages
- Senior roles: near-zero callback rate
- No review/approval before sending
- Damages professional reputation

### Pricing & Trust
- $29/mo DOESN'T include auto-apply (headline feature)
- Non-refundable policy
- BBB F rating for unresolved complaints
- Trustpilot flagged for suspicious reviews

## Features to Replicate

1. Free ATS scanner (no login) — acquisition funnel
2. JD paste-to-tailor workflow
3. Chat-based iteration for cover letters
4. Match percentage scoring
5. Application tracking dashboard

## Features to Skip

1. Auto-apply — low quality, reputation risk
2. Mass application — wrong strategy for senior roles
